# DCSLab

DCSLab - Laravel API

## Installation

Create .env file

>`$ copy .env.example .env`

Fill the required config in .env file

eg database config:
> DB_CONNECTION=mysql  
> DB_HOST=127.0.0.1  
> DB_PORT=3306  
> DB_DATABASE=laravel  
> DB_USERNAME=root  
> DB_PASSWORD=

Run the installation scripts

>`$ php artisan app:install`

## Run Dev Server

Use artisan serve for dev Server

>`$ php artisan serve`

## App Helper

>`$ php artisan list app`
